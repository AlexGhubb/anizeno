Anizeno

    App gestora de series Anime para Android - Proyecto Final Curso
    Autor: Alejandro Soria Torres 
    Tutor del Proyecto: José Antonio Ortega Hidalgo

    Explora listas de series interesantes, guarda tus propias listas, haz búsquedas por título y añade reseñas
    
    La App Anizeno está desarrollada en kotlin Android con Jetpack Compose con el entorno de Android Studio y 
    reuniendo lo aprendido durante el curso sobre base de datos, uso de API, autentificación con Firebase y en general
    el desarrollo en Android actual en este caso con Compose y Material 3.
